# 客户端

也就是在玩 osu! 时，需要运行的程序。

## 版本

| 英文 | 中文 | 备注 |
| :-- | :-- | :-- |
| Stable | 稳定版 | 每月更新 |
| Beta | 候选版 | 每周更新 |
| Cutting edge | 测试版 | 不定期更新 |
| osu!stable (stable) | **不译** |  |
| osu!lazer (lazer) | **不译** |  |

## 界面

| 英文 | 中文 | 备注 |
| :-- | :-- | :-- |
| Overlay | 覆盖层，叠加层 | 根据语境，有时可简译为**界面，层** |
| Screen | 屏幕，*界面，页面* | 根据语境决定[^context-screen] |

[^context-screen]: 现有的例子：`Result screen` -> `结算屏幕`，`Song selection screen` -> `选歌界面`
