## 乐理 {#music}

音乐理论 (**Music Theory**)，详见 [wiki](https://osu.ppy.sh/wiki/zh/Music_theory)

| 英文 | 中文 | 备注 |
| :-- | :-- | :-- |
| Beat | 拍，拍子，节拍 | 对应时间线白线间距 |
| Downbeat | 强拍，下拍 | 通常是小节的第一拍 |
| Upbeat | 弱拍，上拍 | 通常跟在强拍之后 |
